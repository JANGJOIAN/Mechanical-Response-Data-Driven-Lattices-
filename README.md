# Mechanical-Response-Data-Driven-Lattices

This repository contains the code, data, and models used in the paper.

## Repository Structure

- data: training_datasets 
- notebook: Jupyter notebook
- model: trained ML model
- figures: figures used in paper

## How to Use

1. Download the Jupyter notebook (`Main_Code.ipynb`) and training_datasets file from this repository.
2. Place the notebook and the training_datasets in the same working directory on your computer.
3. Launch Jupyter Notebook or JupyterLab.
4. Open `Main_Code.ipynb`.
5. Verify that the file paths in the notebook match the locations of the dataset files.
6. Run cells sequentially to reproduce the results presented in the paper.

Note:
    The "Preparing Data" and "Reading Data" sections are only required if you want to generate the training dataset from the raw data.
    If you have already downloaded the provided training_datasets files, you can skip these two sections. Simply place the training dataset Excel files in the         appropriate directory and run the remaining cells of the Python notebook to reproduce the results.
